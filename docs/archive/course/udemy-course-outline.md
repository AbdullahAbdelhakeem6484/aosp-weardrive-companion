# Udemy Course Outline

## Course Title
Building an AOSP Wear Companion Platform Project with AOSP 16 and Cuttlefish

## Target Audience
- Android platform engineers
- embedded Android developers
- AAOS developers expanding into Wear OS and cross-device systems
- senior Android app engineers moving down the stack

## Module Sequence
1. Project Overview and End Demo
2. Ubuntu Environment and AOSP 16 Workflow
3. Structuring a Non-AOSP Companion Workspace
4. Designing the System Architecture
5. Adding a Custom Android System Service
6. Manager APIs, AIDL, and Privileged Permissions
7. Building the Companion System App
8. Designing a Mock Bluetooth Transport for Emulator-First Development
9. Audio Note Capture and AI Summary Integration
10. Testing, Logs, and Demo Automation
11. Public Demo Packaging and Portfolio Strategy
12. Roadmap to Real Hardware

## Recording Split
- Slide-led lessons: architecture, planning, tradeoffs, roadmap
- Live lessons: AOSP code changes, builds, Cuttlefish launch, app demo, logs, scripts
- Hybrid lessons: AI integration, testing strategy, manager-ready project framing

## Premium Differentiators
- honest emulator limitations
- system-service-first design
- reusable implementation checklists
- public-demo and career-branding guidance
